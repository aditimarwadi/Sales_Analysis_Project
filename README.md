# Sales Trend & Time-Based Performance Analysis

## Project Overview
This project analyzes sales data of a coffee retail business (Afficionado Coffee Roasters) to understand time-based sales patterns. The goal is to identify peak hours, best-performing days, and store-level performance using data analysis and visualization.


## Objectives

### Primary Objectives
- Identify overall sales trends
- Determine peak sales hours
- Find best-performing days

### Secondary Objectives
- Compare store performance
- Analyze product category sales
- Support business decision-making


## Dataset Description
The dataset contains transaction-level data including:
- transaction_id
- year
- transaction_time
- transaction_qty
- store_location
- product_category
- unit_price


## Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit


## Features
- Total Sales & Average Sales calculation
- Yearly Sales Trend visualization
- Hourly Sales Trend analysis
- Day-wise performance
- Store-wise comparison
- Product category analysis
- Interactive filters (Store & Day)


## Streamlit Dashboard
The project includes an interactive dashboard where users can:
- Filter by store location
- Filter by day
- View dynamic charts


## How to Run

1. Install dependencies:
```
pip install pandas
pip install matplotlib
pip install streamlit
```

2. Run the app:
```
streamlit run app.py
```



## Results / Insights
- Thursday is the highest sales day
- Peak sales hour is around 10 AM
- Hell's Kitchen is the best-performing store
- Sales are higher during morning hours
